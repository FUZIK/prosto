# Самиздат-ПРОСТО 
Прила <<ПРОСТО>> для Android для котиков которые не желают посещать злоебучий(кривой/косой как избушка) [сайт](https://%D0%BF%D1%80%D0%BE%D1%81%D1%82%D0%BE%D1%81%D0%BF%D0%B1.%D1%80%D1%84/) для регистрации


## Установка
- для Android - [актуальная версия](https://github.com/FUZIK/prosto/raw/master/androidApp/release/androidApp-release.apk)

## Тех. характер
#### Android 
- compose, coil, brik
#### Kotlin Multiplatform
- навигация, контроллеры, useкейсы, репы, парсер, sqldelight
